Restore App

May 2020

This is a project I did in the UI/UX Bootcamp course. The goal of this app is to help people reuduce, reuse, and recycle.

The app tracks and reminds the user when and where to recycle, ways to reuse products, and has a scanner function to give the user information about the product and suggest possible substitutes. My group also included friends on the profile for the purpose of gamification and to encourage users to use the app and practice eco-friendly habits.
