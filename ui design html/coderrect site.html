Coderrect Site

December 2020

This is a site to report races in codes. The company which is running this is a startup called Coderrect. They had originally given me a mock-up site they had created which was just skin and bones with minimal styling and asked me to improve it. I cleaned up everything and gave the site some style and improved the user interface.

They were having issues with long lines of code in the boxes, so I included scroll bars for extreme text cases. I also made the pop-ups more intuitive for the user as well as a 'back to top' button for long reports.

The site is still being coded by the team at Coderrect, but I did create a working prototype.
